# Simple Asio Messenger
To use this messenger, follow these steps:

1. Download and compile the source code using CMake *or* download the latest release.
2. Open .exe
3. Choose "Server" or "Client" by typing one of these words in console.
4. Enter your nickname
5. (If server) enter port (any) by which anyone in your local network can access you.
   (If client) enter address (ip:port format, example: 127.0.0.1:15001)
6. Congrats, it works!
